What is Xenohuru?

Xenohuru is a free, open-source Progressive Web App (PWA) that showcases all the tourist attractions found in Tanzania. From hiking Mount Kilimanjaro to exploring Zanzibar's Stone Town, from safari drives in the Serengeti to cultural tours in Chagga villages -- this platform brings Tanzania's wonders to the world.

Our Mission

To make Tanzania's incredible tourism information freely accessible to everyone — travellers, developers, and the global community — through a modern, fast, and beautiful open-source platform.

Open Source

Every line of code is public. Fork it, improve it, build on it. We believe that open collaboration creates the best experiences and helps communities grow together.

What This Platform Shows

Attractions

Hiking, safari, cultural tours, beaches, islands, historical sites and more across Tanzania.

Regions + GPS

Each region and attraction shows its latitude and longitude coordinates for precise navigation.

Live Weather

Current weather conditions and forecasts for each attraction, powered by Open-Meteo API.

Distances

How far each attraction is from popular areas like Dar es Salaam, KIA airport, or Moshi city.

Tanzania Time

Live East Africa Time (EAT, UTC+3) displayed across all pages so you always know local time.

Multi-Language

Language switcher scaffold ready for English, Kiswahili, German, and French translations.

Tech Stack

HTML5 CSS3 JavaScript (ES Modules) Tailwind CSS v4 Splide.js AOS (Animate on Scroll) Lucide Icons PWA + Service Worker Django REST API (Backend) Open-Meteo Weather API
